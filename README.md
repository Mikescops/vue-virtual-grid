# Vue Virtual Grid

This is a test project to provide a virtual grid, with infinite scroll.

## Project setup

```
npm ci
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
